title: CalendarType
---

Identifiers for support calendars.

### Syntax

<pre class="syntax">
'buddhist' | 'gregory' | 'iso8601' | 'japanese' | 'persian'
</pre>

### Values
  - `"buddhist"`
    - Buddhist calendar
  - `"gregory"`
    - Gregorian calendar
  - `"iso8601"`
    - ISO-8601 calendar
  - `"japanese"`
    - Japanese calendar
  - `"persian"`
    - Persian calendar
