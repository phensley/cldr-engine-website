title: DateTimePatternFieldType
---

A single character representing a field in a date.

<pre class="syntax">
'y' | 'M' | 'd' | 'a' | 'H' | 'm' | 's'
</pre>

### Values
  - `"y"`
    - Year
  - `"M"`
    - Month
  - `"d"`
    - Day
  - `"a"`
    - Day period (AM, PM)
  - `"H"`
    - Hour
  - `"m"`
    - Minute
  - `"s"`
    - Second
