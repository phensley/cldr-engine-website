Enumerated type identifying a date field, used in formatting relative times.

### Syntax

<pre class="syntax">
'year' | 'quarter' | 'month' ... 'hour' | 'minute' | 'second'
</pre>

### Values
 - 'year'
 - 'quarter'
 - 'month'
 - 'week'
 - 'day'
 - 'sun'
 - 'mon'
 - 'tue'
 - 'wed'
 - 'thu'
 - 'fri'
 - 'sat'
 - 'hour'
 - 'minute'
 - 'second'
